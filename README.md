# Agetha Bot — Telegram Mini App

Панель управления ботом прямо в Telegram.

## Функции

### Для администраторов:
- 👤 Карточка профиля (userinfo)
- 📋 Книга жалоб и репортов
- ⚙️ Настройки чата (антимат, антиспам, рейд-гард и т.д.)

### Для участников:
- 👤 Карточка профиля
- 📢 Отправка жалоб и репортов

## Как настроить

### Шаг 1: Захостить веб-приложение

Mini App требует HTTPS. Бесплатные варианты:

#### GitHub Pages (рекомендуется)

1. Создайте новый репозиторий на GitHub
2. Загрузите файл `index.html` в репозиторий
3. Settings → Pages → Source: `main` branch
4. Подождите 1-2 минуты
5. URL: `https://username.github.io/repo-name/`

### Шаг 2: Настроить бота

В Telegram напишите боту:

```
/panel set https://ваш-url.github.io/agetha-panel/
```

### Шаг 3: Использование

**Для админов:**
```
/panel — открыть панель управления
```

**Для всех:**
```
/app — открыть профиль / панель
```
